# Trip Planner

A simple month-based trip planner web app that suggests destinations based on the selected travel month.

## Live Demo

https://shubhanalytics.github.io/trip-planner/

## Features

- Select travel month from a centered dropdown
- Toggle destination type: India-only or International
- View recommended destinations for the selected month
- See approximate temperature for each destination
- See estimated expense per person
- Enter number of travelers and view estimated total trip cost range
- View common travel options (flight/train/bus/etc.) for each destination

## Run Locally

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## GitHub Pages Deployment

- Deployment workflow: `.github/workflows/deploy-pages.yml`
- Deploys automatically on push to `main`
- Also supports manual trigger from the Actions tab (`workflow_dispatch`)
